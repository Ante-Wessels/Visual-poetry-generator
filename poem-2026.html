Poem Generator



Creates Visual Poetry at random,
Preconfigured sets: On the Road, Hacker's Delight, ! Amorvita !, Shakescene, You!
personalized love poetry,
...or own input


The Poem Generator is a 1999-2000 Java applet.
As Java on websites is discontinued, use the CheerpJ Applet Runner extension (Chrome, some other browsers).


For an explanation of the fields, see the Introduction Gif


Start visual poetry generator

This opens a new window; wait for the extension to start.
You may like to make it full screen (F11) and scroll down a bit.
Make smaller with same command (F11, or hold ESC).
